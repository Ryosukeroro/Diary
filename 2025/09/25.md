# DM3519を動かした！

参考サイト： https://wiki.seeedstudio.com/damiao_series/

## 手順

基本は上記サイトの通りです。

1.  "Damiao Debugging Tools"をダウンロードする。
2.  USBCANとパソコンを繋ぐ。
3.  その後、ESCとCANとGTRを繋ぐ。
4.  モーターとESCを繋ぐ。センサーの信号線も　※電源切った状態でつなごう

<img width="2355" height="1506" alt="image" src="https://github.com/user-attachments/assets/3b145c9a-fc9a-41fe-bfcb-585bb1502bc0" />
画面左下で英語に表示を変えれる。(英語に帰ると赤色になる) 

Open portを押す。(反応するとClose portになる。緑色)

<img width="2160" height="1323" alt="image" src="https://github.com/user-attachments/assets/132c2715-936e-4f61-9e08-08543a9df423" />
Calibrateを押します。(勝手にモーターが動きます。気合入れましょう。固定とか安全に配慮せよ)
<img width="2358" height="1552" alt="image" src="https://github.com/user-attachments/assets/675d7f28-4408-4c3e-a855-0e2708fa3a4d" />
Set parametersに移行します。

その後、Controll settingsのReadを押します。
そして、Read Paramを押します。
<img width="2354" height="1516" alt="image" src="https://github.com/user-attachments/assets/ff6dc421-aa64-46e9-986d-aa820c36b16f" />
Write paramを押します。
