# DM3519を動かした！

参考サイト： https://wiki.seeedstudio.com/damiao_series/

## 手順

基本は上記サイトの通りです。

1.  "Damiao Debugging Tools"をダウンロードする。
2.  USBCANとパソコンを繋ぐ。
3.  その後、ESCとCANとGTRを繋ぐ。
4.  モーターとESCを繋ぐ。センサーの信号線も　※電源切った状態でつなごう
